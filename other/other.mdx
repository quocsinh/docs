## Cơ chế lưu file dialogue

Trước khi lưu file dialogue, phần mềm sẽ tạo ra một folder "backup" và ghi file vào đó. Sau đó sẽ copy ngược lại ra bên ngoài. Mục đích của việc này nhằm tránh đang ghi file mà bị lỗi (sai data, đang ghi file thì treo máy...). Nếu xảy ra sự cố thì file gốc bên ngoài cũng không bị ảnh hưởng.

## File \_fullvh_extras\_.rpy

![O 1 Pn](/images/o_1.png)

Đây là một loại file đặc biệt, chứa tất cả các text không phải là thoại. Thông thường đây là những text kiểu thống kê, nhiệm vụ... thường xuất hiện trong các dòng game sandbox.

Tuy nhiên, vì trình trích xuất text không thể phân biệt được đâu là text cần dịch, nên riêng file này sẽ xuất hiện thêm biểu tượng DELETE. Nút này nhằm mục đích xóa bỏ text không liên quan gì đến game, giúp game sạch hơn và không gây lỗi nhảm nhí.

![O 2 Pn](/images/o_2.png)

Ngoài ra, phía bên phải sẽ in một loạt các text. Các text này ở thời điểm hiện tại không thể dịch được sang Tiếng Việt do Renpy chưa hỗ trợ các text dạng tag. Do do, chỉ có một cách là thay thế trực tiếp vào file rpy gốc.

Nhấn vào mỗi text sẽ nhảy đến dòng cần thay thế. Copy dòng đó và thay thế text gốc bằng VSCode.
