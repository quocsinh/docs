## Cơ chế lưu file dialogue

Trước khi lưu file dialogue, phần mềm sẽ tạo ra một folder "backup" và ghi file vào đó. Sau đó sẽ copy ngược lại ra bên ngoài. Mục đích của việc này nhằm tránh đang ghi file mà bị lỗi (sai data, đang ghi file thì treo máy...). Nếu xảy ra sự cố thì file gốc bên ngoài cũng không bị ảnh hưởng.
